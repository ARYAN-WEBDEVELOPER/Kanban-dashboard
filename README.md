# 🚀 Kanban Task Board

A simple Trello-like task management app built using React and Vite.

---

## 🌐 Live Demo

👉 https://kanban-dashboard-topaz.vercel.app/

---

## 📌 Features

* Add Task (To Do column)
* Edit Task
* Delete Task
* Move Task (Drag & Drop)
* Priority (High / Medium / Low)
* Search Tasks
* Data saved using localStorage
* Popup messages (Add / Delete / Edit)

---

## 🛠️ Tech Stack

* React.js
* Vite
* JavaScript
* CSS
* dnd-kit (Drag & Drop)

## ⚙️ How to Run

1. Open terminal
2. Run:

npm install
npm run dev

3. Open browser:
   http://localhost:5173

---

## 🧠 How it Works

* Tasks are stored in React state
* Each task has:

  * text
  * status (todo / progress / done)
  * priority
* Drag & drop changes task status
* Data is saved in browser using localStorage
